# AI-Powered Stock Market Intelligence Platform

An end-to-end AI/ML research platform for stock market analysis, prediction,
portfolio optimization, and trading strategy research.

**Status:** Phase 1 — Project Setup

## Disclaimer

This project is for educational and research purposes only. It does not
provide financial advice and does not guarantee investment returns.

## Tech Stack (initial)

- Python 3.12
- Pandas, NumPy
- Scikit-learn, XGBoost
- PyTorch
- FastAPI, React (later phases)
- PostgreSQL (later phases)

## Setup

\`\`\`powershell
python -m venv venv
.\venv\Scripts\Activate.ps1
pip install -r requirements.txt
\`\`\`

\`\`\`
